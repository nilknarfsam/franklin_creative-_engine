# Lyrics Structure — Humanização, Tempo e Dinâmica (VibeShell Fase 3)

Como estruturar a letra para soar humana, teatral e com duração adequada.

---

## Estrutura padrão de dinâmica (VibeCore FCE)

```
Intro íntima → beat entra → Pre-Chorus (tensão) → Chorus (explosão)
→ Verse 2 → Chorus → Bridge (breakdown) → Outro (épico) → [End]
```

Espelha o arco musical: **íntimo → arena → íntimo → majestade contida**.

---

## Gestão de tamanho — Super Edição (textos longos)

Para textos longos (ex.: um Salmo inteiro):

1. **Extrair a essência** — não musicar verso a verso
2. **Transliteração rítmica** — criar métrica de rima (AABB, ABAB)
3. **Refrão inédito** — hook explosivo e contagiante, memorável em < 10s
4. Preservar fidelidade ao texto-base (bíblico ou tema)

---

## Alongamento forçado (textos curtos)

Para bater a meta de duração (>3 min):

| Técnica | Tag |
|---------|-----|
| Intro estendida | `[Extended Atmospheric Intro]` |
| Interlúdio instrumental | `[Long Instrumental Interlude]` (>20s) |
| Refrão duplo final | `[Double Chorus]` |
| Espaçamento silábico | reticências `...` |

---

## Quebra de fadiga

Inserir contrastes drásticos no mesmo arranjo:

- Estrofe `[Acoustic, slow]` → refrão `[Massive stadium rock explosion]`
- Queda abrupta para silêncio → explosão pesada
- Alternar densidade instrumental por seção

---

## Momento de ministração (Spoken Word)

Apenas quando aprovado no roadmap (Fase 1):

```
[Spoken Word / Emotional Ministration - Soft acoustic background,
spoken preaching voice, extremely slow, intense build-up]
```

Isolar o instrumental para dar clareza à locução falada. No FCE, tipicamente no **bridge** (clima confessional).

---

## Regras de letra FCE (obrigatórias)

- [ ] 1ª linha: `[Language: Brazilian Portuguese]`
- [ ] Tags de seção em todas as partes
- [ ] Call-and-response marcado com parênteses
- [ ] Pausas com `...`, intensidade com `!`
- [ ] Refrão identificável e repetível
- [ ] Fidelidade ao Salmo/tema base
- [ ] `[End]` presente
- [ ] Sem mistura de idiomas na letra (tags em inglês são exceção)
