# Emotion Engine — Narrative Engine (FCE)

O Emotion Engine traduz **estados da alma** em decisões de câmera, luz, cor, ritmo, música e performance. Emoção não é adjetivo no prompt — é **sistema de coordenadas** para o diretor e para o Prompt Composer.

Cada emoção abaixo define **7 dimensões** + sensação do espectador.

---

## Como usar

1. Identificar emoção **primária** da cena (uma dominante).
2. Opcional: emoção **secundária** (tensão interna).
3. Aplicar tabela nas 7 dimensões.
4. Passar `emotion_id` ao Scene Builder e Composer.

---

## Esperança (`hope`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Slow push-in ou crane up; planos abrem |
| **Iluminação** | Golden hour, backlight, god rays suaves |
| **Cor** | Âmbar, ouro, branco quente; shadows lift |
| **Ritmo** | Cortes mais longos; respiração |
| **Música** | Major, crescendo orgânico, espaço |
| **Personagem** | Ombros baixam; olhar ao horizonte; passos firmes |
| **Espectador** | "Vai ficar bem — não porque é fácil, mas porque há futuro" |

**Escritura:** Sl 37:5, 39:7; Rm 15:13.  
**Cinema:** Final de *The Shawshank Redemption* (chuva e braços abertos).

---

## Arrependimento (`repentance`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Close imóvel ou slow dolly in; ângulo levemente baixo |
| **Iluminação** | Chiaroscuro; key lateral dura; pouco fill |
| **Cor** | Desaturado, azul-verde, baixo contraste |
| **Ritmo** | Cortes lentos; pausas |
| **Música** | Menor, sparse, piano ou cordas solo |
| **Personagem** | Cabeça baixa, mãos abertas ou cobrindo rosto |
| **Espectador** | "Reconheço minha falha — há caminho de volta" |

**Escritura:** Sl 38, 51:17; Lc 15:18.  
**Cinema:** Confession booth lighting (*Hail, Caesar!*), kneeling in rain.

---

## Medo (`fear`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Handheld, Dutch angle raro, claustro |
| **Iluminação** | Underlight, sombras no rosto, flicker |
| **Cor** | Cyan, preto esmagado, highlights frios |
| **Ritmo** | Cortes irregulares ou hold prolongado (tensão) |
| **Música** | Dissonância, sub-bass, silêncios abruptos |
| **Personagem** | Olhos widened, respiração rápida, recuo |
| **Espectador** | "Algo está errado — perigo real ou interior" |

**Escritura:** Sl 23:4 vale; Mt 14:30.  
**Cinema:** Corredor *The Shining* (FCE: medo existencial, não horror gore).

---

## Alegria (`joy`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Wide, orbit, steadicam energético |
| **Iluminação** | High key, sol pleno, poucas sombras duras |
| **Cor** | Saturação moderada-alta, quentes |
| **Ritmo** | Cortes no beat, movimento |
| **Música** | Major, ritmo dançante, brass (VibeCore festivo) |
| **Personagem** | Sorriso genuíno, gestos amplos, palmas |
| **Espectador** | "Celebrar sem culpa — presente de Deus" |

**Escritura:** Sl 47:1, 126; Fp 4:4.  
**Cinema:** Wedding dance *The Godfather* (sem violência), *La La Land* sunrise.

---

## Paz (`peace`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Static ou float muito lento |
| **Iluminação** | Soft diffused, wrap light, sem harsh shadows |
| **Cor** | Pastel, verde-água, creme, lavanda suave |
| **Ritmo** | Long takes, dissolves |
| **Música** | Ambient, acoustic pluck, espaço entre notas |
| **Personagem** | Respiração visível lenta, mãos relaxadas |
| **Espectador** | "Descanso que não depende de circunstância" |

**Escritura:** Sl 37:7, 46:10; Jo 14:27.  
**Cinema:** *Tree of Life* grass scenes; still water reflection.

---

## Vitória (`victory`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Crane up, epic wide, slow motion opcional |
| **Iluminação** | Strong backlight, lens flare controlado |
| **Cor** | Ouro, contraste alto, céu dramático |
| **Ritmo** | Build → release; hold no clímax |
| **Música** | Chorus arena, choir, percussão épica |
| **Personagem** | Postura ereta, olhar up ou forward |
| **Espectador** | "O mal não venceu — o Rei venceu" |

**Escritura:** Sl 40:2-3; 1Co 15:57.  
**Cinema:** *Rocky* steps (secular parallel); *Lord of the Rings* sunrise after battle.

---

## Silêncio (`silence`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Static, minimal movement |
| **Iluminação** | Low key ou single source |
| **Cor** | Quase monocromático |
| **Ritmo** | Sem corte por 3–8 s |
| **Música** | Ausência ou drone mínimo |
| **Personagem** | Imobilidade escolhida, não paralisia |
| **Espectador** | "Santo espaço — Deus fala aqui" |

**Escritura:** Sl 46:10; Hb 2:20.  
**Cinema:** *2001* monolith moments; prayer without score.

---

## Dor (`pain`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Close, handheld, intimate uncomfort |
| **Iluminação** | Cold, top or side hard |
| **Cor** | Blue-gray, crushed blacks |
| **Ritmo** | Staccato ou opressivo long hold |
| **Música** | Minor, dense, operatic pain (emo-rap teatral) |
| **Personagem** | Tensão mandíbula, lágrimas contidas ou corpo curvado |
| **Espectador** | "Isso dói — não estou sozinho" |

**Escritura:** Sl 38, 42:3; Jó 3.  
**Cinema:** *Manchester by the Sea* stillness of grief.

---

## Confiança (`trust`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Steadicam follow confiante; eye-level |
| **Iluminação** | Warm natural, consistent key |
| **Cor** | Terra, verde, âmbar estável |
| **Ritmo** | Moderado, forward momentum |
| **Música** | Groove confiante, trap cadenciado suave |
| **Personagem** | Mãos abertas, passos deliberados, olhar calmo |
| **Espectador** | "Posso entregar — não controlo tudo" |

**Escritura:** Sl 37:3-5; Pv 3:5-6.  
**Cinema:** *Cast Away* raft horizon; father walking child.

---

## Adoração (`worship`)

| Dimensão | Direção |
|----------|---------|
| **Câmera** | Low angle suave (reverência sem caricato) ou wide celestial |
| **Iluminação** | Volumetric light, rays, glow controlado |
| **Cor** | Ouro, roxo profundo, branco puro (paleta FCE) |
| **Ritmo** | Swell visual sync chorus |
| **Música** | Choir, brass, epic hook |
| **Personagem** | Postura de entrega — não performance de palco |
| **Espectador** | "Maior que eu — e isso é bom" |

**Escritura:** Sl 95:6; Ap 4.  
**Cinema:** *Arrival* encounter light; hands open to sky (*Gravity* finale tone).

**FCE:** Evitar palco de igreja clichê — adoração em cotidiano ou natureza épica.

---

## Matriz emoção × hook

| Emoção | Hook ideal |
|--------|------------|
| Medo / Dor | Conflito, Silêncio |
| Confiança / Paz | Contraste, Silêncio |
| Vitória / Alegria | Cold open, Imagem impactante |
| Arrependimento | Pergunta, Close |
| Adoração | Imagem impactante, Contraste luz |
| Esperança | Contraste, Imagem |

---

## Template — Emotion Block (para Composer)

```yaml
emotion_id: trust
primary: trust
secondary: pain
entry_state: anxious
exit_state: peaceful
camera_modifier: "push-in slows, handheld stabilizes"
light_modifier: "cool to warm 3200K over 8 seconds"
color_modifier: "desaturated +15% saturation on skin highlights"
```
