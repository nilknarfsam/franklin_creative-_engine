# Movimentos de Câmera — Veo 3 (FCE)

Vocabulário padronizado para prompts Veo. Usar termos em **inglês** no prompt final.

---

## Enquadramentos (framing)

| Termo | Descrição | Uso FCE |
|-------|-----------|---------|
| `extreme wide shot` | Paisagem dominante, personagem pequeno | Salmo 46 — montanhas; Sl 48 — Sião |
| `wide shot` | Corpo inteiro + ambiente | Celebração, multidão |
| `medium shot` | Cintura para cima | Versos narrativos, rap flow |
| `medium close-up` | Peito/cabeça | Emoção, ministração |
| `close-up` | Rosto | Lamento, clamor, intro íntima |
| `extreme close-up` | Olhos, mãos, detalhe simbólico | Lágrimas, semente, trono |

---

## Movimentos (camera moves)

| Termo | Efeito dramático | Seção musical |
|-------|------------------|---------------|
| `static shot` | Peso, reverência, pausa | Bridge, "aquietai-vos" |
| `slow push-in` | Intimidade crescente | Intro, pre-chorus |
| `slow pull-back` | Revelação, contexto | Outro, resolução |
| `tracking shot` | Urgência, narrativa | Verse rápido |
| `dolly left/right` | Tensão lateral | Conflito, traição |
| `crane up` | Ascensão, triunfo | Chorus arena |
| `crane down` | Humildade, descida ao pó | Lamento Sl 38 |
| `handheld subtle` | Raw, emocional | Emo-rap, clamor |
| `orbit around subject` | Épico, coroação | Sl 45, Sl 48 |
| `aerial drone shot` | Escala divina | Refúgio, cidade santa |
| `steadicam follow` | Jornada, peregrino | Sl 39 viajante |
| `Dutch angle` | Desorientação (usar raro) | Opressão Sl 44 |

## Lentes e profundidade

| Termo | Efeito |
|-------|--------|
| `shallow depth of field` | Isolamento emocional, bokeh |
| `deep focus` | Ambiente narrativo rico |
| `anamorphic lens flare` | Epic arena, pop teatral |
| `rack focus` | Shift personagem → símbolo |
| `soft diffusion` | Memória, fragilidade Sl 39 |

## Ritmo de corte (para Filmora — pós-Veo)

Veo gera clips contínuos; o editor corta entre takes.

| Vibe | Duração média por take | Estilo de corte |
|------|------------------------|-----------------|
| Dark Trap | 3–5s | Cortes secos no snare |
| Arena Rock | 2–4s no chorus | Cortes rápidos + wide holds |
| Lofi / Ambient | 6–8s | Cortes lentos, dissolves |
| Festivo | 2–3s | Cortes no beat + palmas |

## Combinações recomendadas por VibeCore

### Trap Acústico & Pop Teatral (Track 01)
- Intro: `slow push-in`, `medium close-up`, janela/chuva
- Chorus: `crane up`, `wide shot`, golden light burst

### Dark Trap & Emo-Rap (Track 02)
- `handheld subtle`, `close-up`, chiaroscuro, rain streaks
- Evitar movimento excessivo — claustrofobia

### Power Ballad (Track 06)
- `slow push-in` em água/corrente
- `static wide` no chorus com `epic strings` visual (névoa, luz)

### Grand Finale (Track 12)
- `aerial drone` → `orbit` → `crane up` no trono/Sião
- `pull-back` final com `warm acoustic` visual (campo ao amanhecer)

## Frases prontas para colar

```
Slow cinematic push-in toward the subject's face, shallow depth of field.
```

```
Epic crane up revealing vast landscape, anamorphic lens flare, golden hour.
```

```
Static medium shot, minimal movement, dramatic chiaroscuro lighting.
```

```
Smooth steadicam follow from behind, walking through misty forest path.
```

```
Handheld subtle close-up, emotional rawness, rain on window reflection.
```
