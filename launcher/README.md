# FCE Launcher — Índice de tarefas

**Porta de entrada:** [FCE.md](../FCE.md) na raiz do repositório.

Cada arquivo desta pasta é um **playbook conversacional** para IAs (ChatGPT, Cursor, Codex, Claude, Gemini) conduzirem uma tarefa específica do Franklin Creative Engine.

---

## Como usar

1. Humano ou IA abre [FCE.md](../FCE.md)
2. Usuário escolhe opção do menu
3. IA abre o launcher correspondente **abaixo**
4. IA segue ordem de leitura, faz perguntas, consulta arquivos, entrega saída documentada

**Regra:** O launcher orienta a conversa — **não substitui** AGENTS.md, WORKFLOW.md ou arquivos da faixa.

## Estrutura padrão de cada launcher

1. Objetivo da tarefa
2. Ordem de leitura obrigatória
3. Perguntas ao usuário
4. Arquivos a consultar
5. Saídas esperadas
6. Checklist de qualidade
7. O que a IA não deve fazer

---

## Compatibilidade

Funciona em **qualquer chat com acesso ao repositório** — cole o conteúdo de `FCE.md` + launcher se o agente não tiver acesso a arquivos locais.
